**frizzle** is a Python package for combining spectra by forward modeling, as described in `Hogg & Casey (2024) <https://arxiv.org/abs/2403.11011>`_.

The package provides efficient algorithms for spectral combination using Fourier-based methods with JAX for fast computation.
